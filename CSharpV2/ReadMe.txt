New Features in C# 2
---------------------------------------------
1. Generics
2. Partial types
3. Anonymous methods
4. Iterators
5. Nullable types
6. Getter/setter separate accessibility
7. Method group conversions (delegates)
8. Co- and Contra-variance for delegates
9. Static classes
10. Delegate inference
